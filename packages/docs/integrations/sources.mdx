Data sources define the way you send data to Dittofeed, so that you can segment your users and send them personalized messages.

## Available Data Sources

- [Dittofeed's REST API](/integrations/sources/api): Send data directly to Dittofeed's REST API, with [SDK's](/integrations/sdks) available for convenience.
- [Segment.com Integration](/integrations/sources/segment): Use segment.com's customer data platform (CDP) to send data to Dittofeed.
