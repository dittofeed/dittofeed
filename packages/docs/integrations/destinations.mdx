---
title: Data Destinations
description: Send third party applications data from your Dittofeed workspace.
---

Data destinations represent third party applications which Dittofeed can send data to. This data can be used to power analytics, marketing, and other business processes.

## Available Data Destinations

- [Hubspot](/integrations/destinations/hubspot): This Hubspot destination allows you to sync segments as Hubspot lists, and user emails as Hubspot engagements.
- [CSV](/integrations/destinations/csv): Download user segment assignments in a csv file.
