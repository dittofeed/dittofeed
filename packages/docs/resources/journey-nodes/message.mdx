---
title: "Message Journey Node"
description: "Used to message users."
---

Message journey nodes are where the action's at. Without message nodes, journeys wouldn't do anything! Currently message nodes can be used to send emails. In the future they will support other channels as well including: SMS, mobile push notifications, in-app messages, and more.
